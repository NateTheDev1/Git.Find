# Git.Find

### What is Git.Find?
*Git.Find* is a simple web app that integrates the use of the GitHub API to search GitHub users and displayes information about their profile, recent repos, and more.

### Built With
- **Jquery** - JS Tool
- **Bootstrap** -  CSS Styles
- **GitHub API** - Database
